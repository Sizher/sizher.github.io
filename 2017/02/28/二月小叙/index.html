二月小叙

發表於 2017-02-28

还没来得及感叹，「啊！新的一年就这样来了，对待人生，新年要有新的表现呀！」，二月就已经到了最后一天。这个 28 天我想我非常满意，其「获得」程度是此前任何一个 28 天都无法比拟的。

这个月做了四场婚礼，每一场都有一些可回忆的点。2月2日那场的新郎只有 22 岁，长得非常帅，出场时唱着「月亮代表我的心」。2月4日大概是我此生最 RUSH HOUR 的一次经历，因为那是开工的第一天，所以这场内江的主持约定的是当日一早前往，主持完便立即返回成都，回公司上班。我提前订好了 9:48 的动车票，到内江 10:38 ，主持完坐 13:28 的动车回成都，时间安排正好。

9:30 \ 醒来。。。绝望。。。
-
9:50 \ 到达东站。。。绝望。。。
-
10:05 \ 打了好几通电话做解释工作，希望内江的婚庆公司能够在现场，做好安抚工作。事实证明，我这几通电话是完全没有必要的，也是十分耽误事的。
-
10:08 \ 轮到排队至改签窗口，发现还有 10:20 出发内江的一趟车次，到站时间 11:28 可以留给我很充裕的时间赶赴婚礼现场。但是，提前 15 分钟停止售票，只能改签 11:20 的车次，12:00 到站内江。
-
12:03 \ 跑出内江北站，婚庆公司已安排好一辆「摩的」在车站门口等候，这个时间点只有摩托车能够不受拥堵。于是，摩的师傅载着我全程时速 100 飙行穿梭在城市中。说实话，整个过程我考虑的只有万一出车祸我应如何保护自己受伤最小，实在太危险了。
-
12:25 \ 到达婚礼现场，全场鸦雀无声，新娘穿着婚纱坐 T 台前的桌上，新郎在一旁抽烟。我深呼吸了一口，低着头径直走到舞台前，谁的眼睛也不敢看。放下包，拿出电脑，插上音频线，接过话筒。
-
12:28 \ 仪式开始。这场婚礼没有事先彩排，导致仪式过程中新郎新娘与我的配合，各种生涩各种生硬。加之放音乐的师傅并无时间沟通，他居然在新郎新娘公布舞台时，播放「Go Big or Go Home」，干。
-
12:40 \ 仪式结束。非常简单，非常尴尬的一场婚礼。放下话筒，收起电脑，关上书包，我甚至没和新郎握一次手，抽一根喜烟。我又径直走到马路边，拦下一辆出租车，赶赴动车站。
-
13:28 \ 动车启动。没吃早餐，没吃午饭，又开始从两百公里外返回成都。
-
15:03 \ 坐在公司，根本不知道这几个小时里究竟发生了什么。

2月6日的婚礼同样在内江，同样是这样的行程，当然，我并没有再次玩心跳。但急于赶动车回成都，我还是没有吃午饭，没和新郎握手，放下话筒就走。坐在返程的动车上，依然觉得一个小时跨越四百公里的来来回回，真他妈梦幻。

2月12日是一场草坪婚礼，新郎新娘都是二婚，新娘看起来也就三十多岁，朋友圈还晒自己的 LOL 游戏战绩。然后，她的儿子 17 岁，她的出场是由儿子牵着自己走上舞台，完成和新郎的交接。（卧槽！这个组合真刺激！）算一算，新娘生他孩子才多大年纪？新娘儿子在交接时，应该叫这个比他妈年纪还小的男人「爸」还是「大哥」？不管怎么说，很乐观很打趣的一对儿，理应得到我的祝福。

这场婚礼让我能记住，当然不是因为这些有趣的事情。这场我是带着猩猩去的，让他放音乐，包括头一天的彩排和布场，都带着他。目的是想让他全场参与一次婚礼的执行，让他明白这个行当看起来不那么难。勾起他对这一行的兴趣，让他在资阳自己开一家婚庆公司，我会给他全部的指导和帮助。无论如何，端起了这个饭碗，我是一个有抱负和梦想的人，对这门营生我有很多点子，所以我需要一个能够由我支配的实操环境。但自负盈亏这种事，我并不想花太多时间在上面。

制造2月2日这一出「RUSH HOUR」的直接原因，是头一天我花了 16 个小时坐在电脑前到次日 1 点，学习搭建博客网站。做为一个技术小白，从零开始跟着教程学习建站，着实掏空了我的脑浆。在程序员朋友刘佳一的帮助下，检查代码 bug ，修改配置数据，检索域名部署，浪费了人家一个下午。

最终呈现的就是现在这个网站，对于技术党来说，不过是十分基础的把戏。这次上手操作，我实实在在体会了「科技才是第一生产力」这个真谛。IT 技术真的很神奇，撸代码是一件非常有意思的事情，我甚至萌生了学习一门编程语言的想法。Any way，我会不会花心思去学，能否坚持得下去，谁知道呢。不过，对于 IT 的兴趣和关注，我只会增多不会减少。

这个月读完了《月亮与六便士》，花了大约三周时间。阅读几乎都发生在上下班乘坐交通工具的通勤时间，或者睡前做为枕边读物。我正在践行自己的一项坚持，减少对手机的依赖，减少对网络上碎片信息的兴趣。即便地铁上再拥挤，也找缝隙举起书，不低头看手机。一是保护视力，二是拯救愈发弯曲前倾的颈椎。

这本书早已是世界文学的经典，无需再对其赞赏一番。毛姆无疑是讲故事的高手，书中查理斯斯克利特兰德这个人物的精神，令人颇有启发。我是否也有那样一份执着，是否也应该拥有那样的执着。做一个奇怪的人，做一个对梦想对心中「那个东西」追求到偏执的人。无法自己判断会否变成那样，但我不惮那般。

2月16日参加了一场英语考试，距离上次正式考英语已远隔六年之久。虽然备考时间充裕，但我并未参加专门的针对性培训，妄图倚靠自学，当一只固执的烤鸭。事实上，真他妈的想多了。我并不是一个自律性极高的人，何况想从根本上提高英语水平，脱离系统的学习指导，做梦去吧。于是，我还是吃着老本完成了一场「雅思考试」，虽然时至今日也未出成绩，but，go fuck myself。

最后说到运动，整个二月，我并未缺席任何一场篮球的邀约。如果一个人想要对自己有一些改变，那么会体现在他所做的任何事中。我想说是的，新年所打的每一场球，都和去年不同了。之前打球我拒绝一切身体对抗，犹豫持球主动进攻，打球像个姑娘。我怀疑自己是否还行，能否继续进行丢掉多年的运动，就像我也曾怀疑是否还能写出文章。

人所有的烦恼都源自三个字，想太多。瞻前顾后不如放手去做，唯唯诺诺除了后悔什么都得不到。我的意思是，去他妈的鸡汤，放手去做就行了。义无反顾，再来一场装腔作势的主持，再写一篇喃喃自语的文章，再射一个准星未知的中投。就是干，就是干，就是干。

真是巧，文章收笔时 Spotify 来到《Hall of Fame》，好一首鸡血鸡汤鸡油歌。

啊～三月，你别对我太好。
